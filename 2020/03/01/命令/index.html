Linux指令

查询java端口号

Jdk自带的jps命令

$ jps

linux系统提供的ps命令

$ ps -ef | grep java

查看jvm的GC情况

jstat [Options] vmid [interval] [count]
参数说明：
Options，选项，我们一般使用 -gcutil 查看gc情况
vmid，VM的进程号，即当前运行的java进程号
interval，间隔时间，单位为秒或者毫秒
count，打印次数，如果缺省则打印无数次
